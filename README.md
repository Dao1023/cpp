# cpp

大学四年（2021–2025）C/C++ 学习与练习代码归档。从 C 语言入门到数据结构、算法分析、考研刷题、辅导教学，按时间段组织。

## 目录结构

```
src/
├── Projects/          # 早期项目实战
│   ├── Contacts/      # 通讯录（struct + 菜单）
│   ├── Calculate/     # 中缀表达式计算器
│   └── exe/           # PE 文件捆绑器 + ChatGPT 协作
├── algorithm/         # 数据结构与算法课程作业
│   ├── Stack/Queue/UDN/HuffmanCoding/
│   └── 八数码 A*、分治、贪心等
├── tutoring/          # 教培辅导代码
│   ├── luogu/         # 洛谷题解
│   ├── task/          # 零散辅导练习
│   ├── WJC/           # 01–32 完整入门辅导课
│   └── tests/         # 配套材料
└── exam/              # 考研 / 复试题
    ├── 851/           # 数据结构考研（520/527/528/604 日期序列）
    └── 858/           # 编号题 + 复试题（神经网络、滑动窗口等）
```

## 时间线

| 阶段 | 时间 | 内容 |
|------|------|------|
| 大一上 | 2021 秋 – 2022 春 | C 入门：通讯录、计算器 |
| 大一下 / 大二上 | 2022 | 数据结构课程：栈、队列、树、图、哈夫曼 |
| 大二下 / 大三 | 2023 | 算法分析 + ChatGPT 协作 |
| 教培 | 2023 – 2024 | 洛谷、辅导练习、WJC 辅导课 |
| 考研 | 2024 | 851 数据结构 + 858 刷题 |
| 复试 | 2025 | 858 复试题 + 项目整理 |

## 编译

`.vscode/` 已配置 MinGW g++ 编译，输出到 `bin/`。不支持中文路径下的中文目录/文件/变量，需要中文支持请用 MSVC。

```
bin/   # 编译产物（已 gitignore）
build/ # 中间文件（已 gitignore）
```
